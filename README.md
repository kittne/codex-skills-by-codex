# Codex Skills by Codex

A curated set of ChatGPT Codex-ready skills for `.codex/skills`.

## What this repo is
- ChatGPT Codex-ready skills for `.codex/skills`
- Skills generated automatically from ChatGPT Pro Deep Research
- Daily reference and guidance refreshes via Context7

## Usage
1. Clone this repository.
2. Copy the skill folders into your local `.codex/skills` directory.
3. Run Codex with the skills available in that path.

## Update cadence
- Skills and references are refreshed daily using Context7-backed research.

## Repository layout
- One folder per skill
- Each skill contains a `SKILL.md`
- Optional skill assets may include `references/`, `scripts/`, and `agents/`
