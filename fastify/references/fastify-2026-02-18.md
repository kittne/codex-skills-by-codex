## Context7 Sources
- Library: `/fastify/fastify`

## Practical Guidance
- Build by plugin encapsulation, not monolithic route files.
- Define JSON schemas for request/response contracts.
- Use lifecycle hooks (`onRequest`, `preValidation`, `preHandler`, `onResponse`) intentionally.
- Standardize global error handling and status mapping.
- Validate non-functional concerns: timeouts, body limits, graceful shutdown.

## Core APIs
- `fastify.register()` for composition.
- `fastify.addSchema()` and `$ref` reuse for schema governance.
- `fastify.setErrorHandler()` for consistent failure responses.
